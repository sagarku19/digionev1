The real current state of the 4 editors

Three of four are bespoke. The "shared" component only powers the simplest one.

What each editor actually uses today
Editor Shell Tabs Lines Tab layout
linkinbio Bespoke (its own) 5 ~1517 Mini app-sidebar + horizontal tabs
main Bespoke (its own) 9 ~1065 Vertical tab rail
singlepage Bespoke — near-twin of main 9 ~1000 Vertical tab rail
payment SiteVisualEditor (only user) 5 ~116 Collapsible vertical rail

What this means

  • main and singlepage are ~90% duplicated code (same header, undo/redo, theme toggle, device switcher, preview, framer-motion) — just different tab content.
  • linkinbio is a third, different shell with an extra mini app-sidebar and tabs across the top.
  • SiteVisualEditor (the supposed "shared" shell) is only used by payment, the simplest one.
  • Every editor hardcodes per-tab colors (blue/emerald/rose/amber/fuchsia…) — all violate the dashboard token rule.
